Compreensão do Spark Java:

O Spark Java é um framework web leve e simples para criação de aplicativos web em Java sendo principalmente usado para o desenvolvimento de APIs RESTful, dentre suas várias vantagens podem ser citadas: simplicidade e minimalismo, fácil configuração, independente de servidores e aplicativos externos e sua alta flexibilidade.

Comparação com Outros Frameworks:

O Spark quando comparado a outra API que utilizamos ao decorrer do trimestre (SpringBoot) ele se destaca devido a sua simplicidade na hora das atribuições, mas, em contrapartida, se torna limitado na hora de compara-lo no ponto de vista profissional. O que não o torna necessariamente pior nesse sentido, já que ele pode ser muito útil na criação de aplicativos simples.

Escalabilidade e Desempenho:

O Spark Java suporta operações assíncronas, o que permite lidar com um grande número de solicitações simultâneas de forma eficiente, garantindo uma boa capacidade de resposta e escalabilidade. Devido à sua arquitetura leve e minimalista, o Spark Java impõe pouca sobrecarga ao sistema, o que torna ele muito mais rápido que as outras APIs e devido a sua facilidade ele se torna muito escalável e alterável dependendo da situação.

Facilidade de Uso:

O Spark possui uma API limpa e intuitiva, além de ser muito fácil de se entender, tornando o desenvolvimento mais produtivo, não possui a necessidade de configurações em arquivos externos, pois tudo é feito diretamente por meio de programação e com isso torna o desenvolvimento de aplicações muito mais rápido.
